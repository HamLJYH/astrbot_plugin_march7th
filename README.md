# astrbot_plugin_march7th
一个可以用指令“/三月七”来随机输出三月七句子的插件

> "我的过去或许不在从前，而是在我的未来里。"

一个为 AstrBot 打造的《崩坏：星穹铁道》三月七角色语句插件，让可爱的三月七陪伴你的每一次聊天。

---

## ✨ 为什么选择这个插件？

| 特点 | 说明 |
|------|------|
| 🎯 **专注单一角色** | 只收录三月七，不做大而全的杂烩 |
| 💬 **贴近原作** | 内置台词均来自游戏官方语音和剧情 |
| 🛠️ **高度可定制** | 随时添加、删除、管理自己的金句库 |
| 📦 **开箱即用** | 100+ 条内置金句，安装即用 |
| 💾 **数据持久化** | 自定义内容自动保存，重启不丢失 |

---

## 🚀 快速开始

### 安装

```bash
# 方式一：直接克隆到插件目录
cd /path/to/astrbot/data/plugins/
git clone https://github.com/HamLJYH/astrbot_plugin_march7th.git

# 方式二：手动下载
# 将插件文件夹放入 data/plugins/ 后重启 AstrBot

## 基础使用

用户: /三月七
Bot: 📷 三月七 · 角色语音
     本姑娘就是三月七，如假包换！

## 📖 指令详解

从内置库 + 自定义库中随机抽取一条三月七台词。

📷 三月七 · 剧情台词 · [自定义]
我的过去或许不在从前，而是在我的未来里，所以我一定会一站站走下去，哪怕有一天没有列车。

 /添加三月七金句 <内容> [来源]  — 添加金句
扩充你的三月七语录库。
