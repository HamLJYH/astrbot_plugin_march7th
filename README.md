# astrbot_plugin_march7th
一个可以用指令“/三月七”来随机输出三月七句子的插件

> "我的过去或许不在从前，而是在我的未来里。"

一个为 AstrBot 打造的《崩坏：星穹铁道》三月七角色语句插件，让可爱的三月七陪伴你的每一次聊天。

---

## ✨ 为什么选择这个插件？

| 特点 | 说明 |
|------|------|
| 🎯 **专注单一角色** | 只收录三月七，不做大而全的杂烩 |
| 💬 **贴近原作** | 内置台词均来自游戏官方语音和剧情 |
| 🛠️ **高度可定制** | 随时添加、删除、管理自己的金句库 |
| 📦 **开箱即用** | 100+ 条内置金句，安装即用 |
| 💾 **数据持久化** | 自定义内容自动保存，重启不丢失 |

## 基础使用

用户: /三月七
Bot: 📷 三月七 · 角色语音
     本姑娘就是三月七，如假包换！

---

## 功能

- 随机金句：输入 `/三月七` 随机输出一条金句
- 自定义添加：输入 `/添加三月七金句` 添加自己的金句
- 删除金句：输入 `/删除三月七金句` 删除自定义金句
- 列表查看：输入 `/三月七金句列表` 查看所有自定义金句
- 统计分析：输入 `/三月七金句统计` 查看金句统计

---

## 指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/三月七` | 随机输出一条三月七金句 | `/三月七` |
| `/添加三月七金句` | 添加自定义金句 | `/添加三月七金句 "三月七最可爱啦！" 角色语音` |
| `/删除三月七金句` | 删除包含关键词的自定义金句 | `/删除三月七金句 开拓者` |
| `/三月七金句列表` | 查看自定义金句列表 | `/三月七金句列表 2` |
| `/三月七金句统计` | 查看金句统计信息 | `/三月七金句统计` |
| `/三月七帮助` | 显示帮助信息 | `/三月七帮助` |
