# astrbot_plugin_march7th
一个可以用指令“/三月七”来随机输出三月七句子的插件

> "我的过去或许不在从前，而是在我的未来里。"

一个为AstrBot框架打造的《崩坏：星穹铁道》三月七角色语句插件，让可爱的三月七陪伴你的每一次聊天。


## 功能

- 随机语句：输入 `/三月七 语句` 随机输出一条语句
- 自定义添加：输入 `/三月七 添加` 添加自己的语句
- 删除语句：输入 `/三月七 删除` 删除自定义语句
- 列表查看：输入 `/三月七 列表` 查看所有自定义语句
- 统计分析：输入 `/三月七 统计` 查看语句统计

## 安装

1. 将本插件文件夹复制到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 或热重载插件

## 指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/三月七 语句` | 随机输出一条三月七语句 | `/三月七 语句` |
| `/三月七 添加 <内容> [来源]` | 添加自定义语句 | `/三月七 添加 "三月七最可爱啦！" 角色语音` |
| `/三月七 删除 <关键词>` | 删除包含关键词的自定义语句 | `/三月七 删除 开拓者` |
| `/三月七 列表 [页码]` | 查看自定义语句列表 | `/三月七 列表 2` |
| `/三月七 统计` | 查看语句统计 | `/三月七 统计` |
| `/三月七 帮助` | 显示帮助信息 | `/三月七 帮助` |

本插件指令形成了指令组，如忘记指令，可输入 `/三月七` 来让系统自动显示出补全的指令组。

## 防刷屏说明

用户级防刷屏：
 
开启后，同一用户在指定时间间隔内只能触发一次  /三月七 语句 
 
重复触发会提示剩余冷却时间
 
私聊和群聊共用同一冷却记录

群聊日限：
 
开启后，每个群聊每天最多只能触发指定次数的随机语句
 
达到上限后全群当天无法再触发
 
每天 0 点自动重置
 
私聊使用独立计数（前缀  private_ ）

不受限制的指令： 添加、删除、列表、统计、帮助

## 内置语句

插件内置了 100+ 条三月七经典语句，涵盖：
- 角色语音
- 剧情台词

## 注意事项
 
默认语句无法删除，只能删除自定义语句
 
自定义语句保存在插件目录的  custom_quotes.json  中
 
添加语句时内容必填，来源可选
 
防刷屏配置可在 AstrBot 控制台修改
 
用户冷却和群聊日限仅限制  /三月七 语句  指令
 
记录保留最近一天，过期自动清理

## 自定义语句

自定义语句保存在插件目录的 `custom_quotes.json` 文件中，格式如下：

```json
[
  {
    "content": "语句内容",
    "source": "来源"
  }
]
```
